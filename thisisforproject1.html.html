How many ways to apply CSS?
Trick question, only one correct way: externally baby!
  • link rel="stylesheet" href="styles.css"
    • href: specifies the path to the CSS file
    • type: specifies the type of document being linked to
    • rel: specifies the relationship between the HTML page and the file it is linked to
Selectors:
  • Universal: *
    • Applies to all elements in the document
  • Type: h1, h2, h3
    • Matches element names
  • Class: .className
    • Matches an element whose class attribute has a value that matches the one specified after the period symbol
  • ID: #idName
    • Matches an element whose id attribute has a value that matches one specified after the pound symbol
  • Child: li>a
    • Matches an element that is a direct child of another
  • Descendant: p a
    • Matches an element that is a descendant of another specified element
  • Adjacent Sibling Selector: h1+p
    • Matches an element that is the next sibling of another
  • General Sibling Selector: h1~p
    • Matches an element that is a sibling of another, although it does not have to be the directly preceding element
CSS Cascade:
  • Last Rule: If the two selectors are identical, the latter of the two will take precedence
  • Specificity: If one selector is more specific than the others, the more specific rule will tak precedence over more general ones.

Layout
Positioning schemes:
  • Normal flow: Every block-level element appears on a new line, causing each item to appear lower down the page than the previous one. (Default)
    • position: static;
  • Relative positioning: Moves an element from the position it would be in normal flow, shifting it to the top, right, bottom, or left of where it would have been placed.
    • position: relative;
      • To move up or down use top or bottom properties
      • To move left or right use left or right properties.
  • Absolute positioning: This positions the element in relation to its containing element.
    • position: absolute;
Box offset:
  • Fixed positioning: A form of absolute positioning that positions the element in relation to the browser window.
    • position: fixed;
    • z-index: num;
  • Floating elements: Allows you to take that element out of normal flow and position it to the far left or right of a containing box.
    • float: left, right;
    • clear: ;
    • To create multi-column layouts use float, width, and margin.
